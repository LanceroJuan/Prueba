# Cálculos de Cortocircuito

Scripts y hojas de cálculo para el análisis de corrientes de cortocircuito en sistemas eléctricos.

## Contenido esperado

- Cálculo de corrientes de falla trifásica, bifásica y monofásica (métodos IEC 60909, ANSI/IEEE)
- Determinación de impedancias de secuencia (positiva, negativa y cero)
- Cálculo de potencia de cortocircuito (Scc) en barras
- Verificación de capacidad de ruptura de interruptores
- Modelos de red para distintas topologías (radial, mallada)

## Formatos

Scripts Python/MATLAB, hojas Excel con macros, o archivos de entrada para software especializado (ETAP, DIgSILENT, PowerFactory).
