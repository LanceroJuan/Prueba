# Cálculos de Protecciones

Herramientas para la coordinación y ajuste de sistemas de protección eléctrica.

## Contenido esperado

- Ajuste de relés de sobrecorriente (51/50) con curvas IEC/IEEE
- Coordinación tiempo-corriente (curvas TCC) para relés, fusibles e interruptores
- Cálculo de zonas de protección diferencial de transformadores y barras
- Ajuste de relés de distancia (21) para líneas de transmisión
- Verificación de sensibilidad y selectividad del esquema de protecciones

## Formatos

Scripts Python, hojas Excel de coordinación, archivos para SEL-5030/AcSELerator, EasyPower o ETAP.
