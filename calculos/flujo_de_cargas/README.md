# Cálculos de Flujo de Cargas

Scripts y modelos para el análisis de flujo de potencia (load flow) en redes eléctricas.

## Contenido esperado

- Implementaciones de métodos iterativos: Newton-Raphson, Gauss-Seidel, desacoplado rápido
- Modelos de generadores, cargas y transformadores
- Análisis de perfiles de tensión y márgenes de estabilidad
- Estudios de contingencias (criterio N-1)
- Optimización de flujo de potencia (OPF)

## Formatos

Scripts Python (pandapower, PyPSA), MATLAB/Simulink, o archivos de caso para PSCAD/PSS/E.
