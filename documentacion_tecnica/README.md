# Documentación Técnica

Base de conocimiento y referencias para los estudios y herramientas del repositorio.

## Contenido esperado

- Notas metodológicas y bases teóricas de los cálculos implementados
- Resúmenes de normas aplicables (IEC, IEEE, RETIE, NTC, etc.)
- Guías de uso de las herramientas y scripts del repositorio
- Glosario de términos y abreviaturas
- Registro de decisiones técnicas (ADR) y criterios de modelado adoptados

## Estructura sugerida

```
documentacion_tecnica/
├── normas/          # Resúmenes y referencias normativas
├── metodologias/    # Fundamentos teóricos de cada tipo de cálculo
└── guias_uso/       # Instrucciones de uso de herramientas del repo
```
