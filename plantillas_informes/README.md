# Plantillas de Informes

Plantillas estandarizadas para la elaboración de informes técnicos de ingeniería eléctrica.

## Contenido esperado

- Plantillas Word/LaTeX para informes de estudios eléctricos (cortocircuito, flujo, protecciones)
- Portadas y hojas de control de revisiones
- Tablas y figuras con formato corporativo
- Plantillas de memorias de cálculo
- Checklists de revisión técnica previos a entrega

## Convenciones de nomenclatura

`INFORME_<TIPO_ESTUDIO>_<PROYECTO>_<REV>.docx`
Ejemplo: `INFORME_CORTOCIRCUITO_SUBESTACION_A_REV2.docx`
