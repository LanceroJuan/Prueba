# Herramientas de Ingeniería Eléctrica

Repositorio centralizado de scripts, plantillas y documentación para estudios de sistemas eléctricos de potencia.

## Estructura del repositorio

```
.
├── calculos/
│   ├── cortocircuito/      # Análisis de corrientes de falla (IEC 60909, ANSI/IEEE)
│   ├── flujo_de_cargas/    # Flujo de potencia y estudios de estabilidad
│   └── protecciones/       # Coordinación y ajuste de relés y fusibles
├── plantillas_informes/    # Plantillas Word/LaTeX para memorias y reportes
├── scripts_automatizacion/ # Automatización de tareas y procesamiento de datos
└── documentacion_tecnica/  # Bases teóricas, normas y guías de uso
```

Cada carpeta contiene su propio `README.md` con detalle del contenido esperado y convenciones aplicables.
