# Scripts de Automatización

Utilidades para automatizar tareas repetitivas en flujos de trabajo de ingeniería eléctrica.

## Contenido esperado

- Procesamiento y limpieza de datos de medidores y SCADA
- Generación automática de reportes a partir de resultados de simulación
- Scripts de exportación/importación entre software (ETAP, DIgSILENT, PSS/E, Excel)
- Automatización de casos de estudio en lote (batch simulations)
- Extracción de datos de planos eléctricos (DWG/DXF)

## Requisitos

Documentar dependencias externas en un archivo `requirements.txt` (Python) o equivalente junto a cada script.
